## Distincția față de celelalte tarife pe direcția Briceni

Acest tarif **NU trece prin Briceni**. Drumul ocolește Briceni:
- Caracușenii Vechi → Corjeuți → **Trinca → Tîrnova → Gordineștii Noi → Edineț** (direct)

Spre deosebire de:
- **Tarif 106/105 (Criva Direct):** Criva → ... → Briceni → Halahora → Edineț
- **Tarif 98/116 (Criva via Corjeuți):** Criva → ... → Corjeuți → Caracușenii Vechi → Tabani → Briceni → Halahora → Edineț

După Edineț, **același drum** ca în tarifele 106 și 98 — aceleași segmente km până la Chișinău.
