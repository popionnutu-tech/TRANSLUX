## Distincția față de celelalte tarife

Acest tarif folosește un **drum complet diferit** între Lipcani și Bălți:
- **NU trece prin Briceni** (ca tarif 106 sau 104)
- **NU trece prin Edineț** (ca tarif 106 sau 115)
- Trece prin sate mici: Bezeda → Lopatnic → Viișoara → Bădragii → Hancăuți → Corpaci → Cuconeștii Noi → Dumeni → Duruitoarea Nouă → Văratic → Druța → Zaicani → Pîrjota
- **Intră în orașul Rîșcani** (alte tarife trec doar prin Intersecția Rîșcani)
- După Intersecția Rîșcani, **același drum** ca în tarifele 106/104/98 (până la Chișinău)
